12-Tone Chromatic Tuner (@ 440 Hz)
JavaScript · HTML / CSS · 2025-2026

Chromatic 12-tone, simple, vanilla web-stack application that functions as a musical instrument tuner and pitch estimation tool, utilizing the YIN pitch detection algorithm and a low-pass filter to capture the bass(iest) note, using that as the determinant tone for the audio pipeline. Operates @ 440 Hz frequency range, i.e. the international tuning standard.
